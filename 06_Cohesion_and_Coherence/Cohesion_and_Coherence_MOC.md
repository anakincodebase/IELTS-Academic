## Band 8.5 Fast Route
1. Master one connector system (do not memorize random lists).
2. Build lexical/reference chains for 3 Task 2 topics daily.
3. Practice paragraph coherence using T-E-E-L (Topic, Explain, Example, Link-back).
4. Record one Part 3 answer daily and audit flow logic.
5. Run final 30-second cohesion and coherence scan before submission/speaking end.

## Exam-Week Deployment
- Writing Task 2: prioritize coherence of argument over complex vocabulary display.
- Writing Task 1: prioritize grouping logic over number dumping.
- Speaking Part 3: prioritize opinion -> reason -> example -> balance sequence.
