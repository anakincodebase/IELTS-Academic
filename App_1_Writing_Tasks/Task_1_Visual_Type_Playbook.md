## Task 1 Reality Check (Band 6→8 Shift)

> [!danger] CRITICAL BAND 7 BLOCKER
> Band 6 describes EVERYTHING they see. Band 8 selects TOP 3-4 PATTERNS ONLY.
> Success = Strategic pattern identification, not exhaustive description.

**What Task 1 Tests:**
- Can you describe visual data accurately? (Not creative writing)
- Can you identify KEY trends/patterns? (Not minor details)
- Can you write with precise professional English? (Vocabulary + grammar)

**Critical Mistake**: Band 6 describes EVERYTHING they see; Band 8 selects TOP 3-4 patterns

**Time Allocation**: 
- 2 min: Analyze visual (identify type + key patterns)
- 18 min: Write 150+ words with structure
- 5 min: Proofread

## THE 7 TASK 1 VISUAL TYPES

### TYPE 1: LINE GRAPHS (Most Common)

**What You'll See**: Two or more lines showing change over time

**Band 8 Strategy**:
```
✓ Identify: What did each line do? (rose/fell/remained stable)
✓ Describe: Turning points (when did direction change?)
✓ Compare: Which line highest/lowest? When did lines intersect?
✓ Avoid: "As you can see on the graph..." (too informal)
```

**Structure for Line Graphs**:
```
Paragraph 1 (Overview): 
"The graph illustrates [what is being measured] between [dates]. 
Overall, [main trend: all lines rose/fell/one rose and one fell, etc.]"

Paragraph 2-3 (Details):
"[Line A] increased from X in [year] to Y in [year], reaching a peak of Z in [year].
Subsequently, [Line B] demonstrated [opposite/similar] trajectory, [specific numbers]."

Paragraph 4 (Comparison/Conclusion):
"By [final year], [line] had surpassed [other line], with the difference widening to Z units."
```

**Band 6 vs Band 8 Language**:

| Band 6 | Band 8 |
|---|---|
| "went up" | "increased/rose/surged/accelerated" |
| "went down" | "declined/fell/plummeted/decreased" |
| "stayed the same" | "remained stable/plateaued/maintained consistency" |
| "The most" | "peaked at/reached a maximum of" |
| "In 2010" | "Throughout the period 2010-2015" / "Between 2010 and 2015" |
| "numbers" | "figures/values/units/data points" |

**50-Word Example (Band 8)**:
```
"The line chart demonstrates electricity consumption across four nations over three decades. 
Overall, consumption escalated significantly in all countries, though at divergent rates. 
Germany and France increased from approximately 400 to 600 units, while consumption in Spain 
and Italy remained comparatively lower, rising from 250 to 400 units respectively."
```

### TYPE 2: BAR CHARTS (Very Common)

**What You'll See**: Vertical or horizontal bars comparing values across categories

**Band 8 Strategy**:
```
✓ Groups: Are there categories? (countries/products/age groups)
✓ Ranking: Order bars from highest to lowest in your description
✓ Patterns: Do similar bars group together? Any outliers?
✓ Comparison: Compare highest vs lowest; compare trends
```

**Structure for Bar Charts**:
```
Paragraph 1 (Overview):
"The bar chart compares [what is measured] across [categories] in [time period].
Overall, [strongest pattern: which category scored highest/lowest]."

Paragraph 2 (Major Categories):
"[Category A] represented the highest value at X, followed by [Category B] at Y.
In contrast, [Category C] demonstrated significantly lower figures, reaching only Z."

Paragraph 3 (Comparisons):
"The disparity between [highest] and [lowest] categories amounted to A units.
[Category B] exceeded [Category C] by approximately B units, indicating..."

Paragraph 4 (Conclusion):
"In summary, [pattern observed across all categories]."
```

**Band 6 vs Band 8 Comparisons**:

| Band 6 | Band 8 |
|---|---|
| "Big" / "Small" | "substantial/considerable" / "marginal/minimal" |
| "Much more" | "approximately X times greater" / "exceeded by Z%" |
| "About/Around" | "approximately/roughly/in the region of" |
| "More/Less" | "considerably/significantly more/less" |
| "[Numbers] and [Numbers]" | "[Numbers], representing Y% of total" |

**50-Word Example (Band 8)**:
```
"The bar chart illustrates annual spending across five departments in 2024. Finance 
received the largest allocation at £4.2 million, representing 35% of total expenditure. 
HR and Marketing received modest allocations of £1.8 and £2.1 million respectively, 
while Operations and IT received approximately £3.2 and £2.7 million."
```

### TYPE 3: PIE CHARTS (Less Frequent but Important)

**What You'll See**: Circle divided into segments, each representing a proportion

**Band 8 Strategy**:
```
✓ Percentages: What percentage each slice? Are any under 5%? Group them
✓ Grouping: Create meaningful categories (large/medium/small slices)
✓ Comparison: What's largest/smallest? How much bigger?
✓ Avoid: Describing every tiny 1% slice (group into "others")
```

**Structure for Pie Charts**:
```
Paragraph 1 (Overview):
"The pie chart demonstrates the composition of [category] in [year/period].
Overall, [largest segment] dominated the distribution at X%, while [smallest segments] 
accounted for significantly smaller proportions."

Paragraph 2-3 (Main Segments):
"[Segment A] represented [X%] of the total, the largest share. 
[Segment B] comprised [Y%], followed by [Segment C] at [Z%].
The remaining segments collectively constituted [remaining%]."

Paragraph 4 (Meaningful Patterns):
"Notably, the top three segments accounted for [combined%] of total expenditure,
suggesting concentration in these areas."
```

**Band 8 Pie Chart Language**:

| Phrase | Usage | Example |
|---|---|---|
| "accounted for" | Portion descriptors | "China accounted for 42% of total production" |
| "constituted" | Formal equivalent of "made up" | "The remaining segments constituted 8%" |
| "represented" | Show proportion | "Transportation represented 23% of spending" |
| "the largest/smallest share" | Extremes | "Housing represented the largest share at 45%" |
| "collectively" | Group multiple segments | "These categories collectively comprised 60%" |
| "the remaining" | Everything else | "The remaining segments totaled 15%" |

**50-Word Example (Band 8)**:
```
"The pie chart illustrates energy source distribution in 2024. Fossil fuels 
dominated at 58%, followed by renewables at 28% and nuclear at 14%. Notably, 
fossil fuels' substantial share reflects continued reliance on conventional energy. 
However, the 28% renewable proportion signals emerging transition toward sustainable sources."
```

### TYPE 4: TABLES (Data-Heavy)

**What You'll See**: Rows and columns of numerical data

**Band 8 Strategy**:
```
✓ Structure: Is it comparing across time? Across categories? Both?
✓ Trends: Do numbers increase/decrease down rows or across columns?
✓ Extremes: Find highest/lowest; describe difference
✓ Patterns: Any consistent patterns? Anomalies?
```

**Structure for Tables**:
```
Paragraph 1 (Overview):
"The table presents data on [what is measured] across [timeframe/categories].
Overall, [strongest pattern: consistent increase/decrease/variation]."

Paragraph 2 (Column/Row Trends):
"Examining [first category], values ranged from X in [year] to Y in [year],
demonstrating an [increasing/decreasing] trajectory. Conversely, [second category] 
fluctuated between A and B, with no consistent pattern."

Paragraph 3 (Comparisons):
"The disparity between [highest value] and [lowest value] totaled Z units.
[Category A] consistently surpassed [Category B] throughout the period."

Paragraph 4 (Conclusion):
"In summary, [pattern that emerges from data]."
```

**Band 8 Table Descriptors**:

| Descriptor | Meaning | Example |
|---|---|---|
| "fluctuated between" | Varied back and forth | "Prices fluctuated between £50-70 throughout Q1-Q4" |
| "peaked at" | Reached maximum | "Sales peaked at 2,500 units in March" |
| "dipped to" / "plummeted to" | Reached minimum | "Revenue dipped to £1.2M in Q3" |
| "consistently surpassed" | Regularly exceeded | "Product A consistently surpassed Product B in sales" |
| "remained relatively stable" | Didn't change much | "Employment figures remained relatively stable at 500k" |
| "ranged from X to Y" | Showed variation | "Temperatures ranged from 15°C to 28°C" |

### TYPE 5: MAPS (Less Frequent)

**What You'll See**: Geographic layout showing locations, changes, or demographics

**Band 8 Strategy**:
```
✓ Layout: Describe spatial relationships (north/south/west/east)
✓ Changes: What's new? What's removed? What's expanded?
✓ Connections: Are areas linked? How?
✓ Accuracy: Use compass directions; avoid vague "here/there"
```

**Structure for Maps**:
```
Paragraph 1 (Overview):
"The two maps illustrate [location] in [year] and [year], showing [type of change].
Overall, [major transformation description]."

Paragraph 2-3 (Spatial Descriptions):
"In the original map, [feature] was located in the [direction] of the site,
adjacent to [neighboring feature]. By [later year], [change that occurred]."

"The [new feature] was constructed to the [direction], spanning approximately X area.
Meanwhile, [existing feature] was [expanded/removed/relocated]."

Paragraph 4 (Overall Impact):
"These modifications transformed the layout substantially, with the site
becoming [more/less] developed and [characteristic of change]."
```

**Band 8 Map Language**:

| Phrase | Usage |
|---|---|
| "to the north/south/east/west of" | Precise location |
| "adjacent to" | Next to |
| "spanning approximately X meters/area" | Size description |
| "extended to encompass" | Expansion description |
| "was relocated to" | Movement |
| "occupied the [direction] portion" | Area positioning |

### TYPE 6: PROCESS DIAGRAMS (Procedural)

**What You'll See**: Steps shown sequentially (linear or cyclical)

**Band 8 Strategy**:
```
✓ Sequence: Describe in order; use sequencing language
✓ Active Voice: Who/what performs the action?
✓ Technical Terms: Use specific terminology (even if simple process)
✓ Cyclical: If circular, explain how it returns to start
```

**Structure for Processes**:
```
Paragraph 1 (Overview):
"The diagram illustrates the process of [what is made/happens], consisting of [number] 
stages. Overall, [process direction: unidirectional/cyclical]."

Paragraph 2-3 (Sequential Steps):
"Initially, [Stage 1] occurs when [input/action]. 
Subsequently, [Stage 2] takes place, wherein [description].
Following this, [Stage 3 is] completed through [method], resulting in [output].
Finally, [Stage 4] represents the [end result/return to start]."

Paragraph 4 (Process Summary):
"In total, the process comprises [number] sequential stages, with each stage building upon..."
```

**Band 8 Process Language**:

| Sequence | Phrase |
|---|---|
| First | "Initially, the process begins with..." / "Stage one involves..." |
| Second | "Subsequently, [step] occurs..." / "Following this..." |
| Third | "Next, [step] is carried out..." / "Thereafter..." |
| Fourth | "Finally, [step] completes..." / "The final stage produces..." |
| Outcome | "As a result of..." / "This yields..." / "The output is..." |
| Throughout | "Throughout the process..." / "At each stage..." |

**50-Word Example (Band 8)**:
```
"The diagram outlines water purification, comprising five stages. Initially, raw water 
enters the filtration system, removing large particles. Subsequently, chemical treatment 
eliminates contaminants. Following this, sedimentation settles remaining particles. Finally, 
chlorination disinfects the water before distribution. The entire process ensures safe, 
potable water supply to consumers."
```

### TYPE 7: MIXED VISUALS (Combined Charts)

**What You'll See**: Two or more chart types combined (e.g., line graph + bar chart)

**Band 8 Strategy**:
```
✓ Separate Analysis: Treat each visual type according to its rules
✓ Connections: How do the two visuals relate?
✓ Synthesis: Combine findings from both into coherent description
```

**Structure for Mixed Visuals**:
```
Paragraph 1 (Overview - Both):
"The charts illustrate [Visual 1 shows X], while [Visual 2 shows Y].
Overall, [combined pattern from both]."

Paragraph 2 (Visual 1 Analysis):
"The [line graph/bar chart] demonstrates [pattern in Visual 1]."

Paragraph 3 (Visual 2 Analysis):
"The [second visual] reveals [pattern in Visual 2]."

Paragraph 4 (Synthesis):
"Collectively, the data suggest [relationship between the two visuals]."
```

## COMMON TASK 1 MISTAKES (Band 6 Indicators)

| Mistake | Why It's Wrong | Band 8 Fix |
|---|---|---|
| "As you can see..." | Too informal/conversational | Cut phrase; start with data directly |
| Describing every detail | Lacks selection/prioritization | Select 3-4 main patterns only |
| "In my opinion..." | Task 1 is objective, not opinion | Remove personal commentary |
| Present tense for past data | "The graph shows sales go up" | Use past: "sales increased" |
| No overview | Jumps to details | Start with clear overview statement |
| Vague numbers | "about lots" | Give specific figures: "approximately 450 units" |
| No comparison | Lists facts independently | Compare: "A exceeded B by 50%" |
| Grammar errors | Subject-verb agreement, articles | Use error tracking: [[Error_Classification_System]] |

## QUICK CHECKLIST: Before Submitting Task 1

- [ ] Overview paragraph explains what visual shows + main trend?
- [ ] 150-170 words minimum?
- [ ] Selected 3-4 key patterns (not described everything)?
- [ ] Used past tense for historical data?
- [ ] Used specific numbers (not vague)?
- [ ] Compared values/trends across categories/time?
- [ ] Used academically precise language?
- [ ] No spelling/grammar errors checked?
- [ ] No personal opinion statements?
- [ ] No "As you can see" / "The graph shows" opening?
