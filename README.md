### Important: How This Works on GitHub vs. Obsidian

This vault was created in Obsidian, which has a different way of handling embedded images and audio than GitHub. Here's what you need to know:

**On GitHub:**
- Images embedded as Obsidian links (`![[image.png]]`) won't display
- Audio files aren't embedded in standard markdown

**How to access the media:**

**Option 1: Best Experience — Use Obsidian**
- Download this repository
- Open it in [Obsidian](https://obsidian.md) (free application)
- All images display inline exactly as intended
- All internal links work perfectly
- Audio files accessible from the sidebar
- This is the experience the vault was designed for


**Option 2: Clone and Use Locally**
```bash
git clone [this-repo-url]
cd IELTS-Academic-Mastery-Vault
```
Then open the folder in Obsidian for the full experience.
