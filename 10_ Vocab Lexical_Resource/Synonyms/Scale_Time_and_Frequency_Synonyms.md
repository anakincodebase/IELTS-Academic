# Scale Time and Frequency Synonyms

## Core Mapping (IELTS-Safe)
| Core Word | Better Alternatives | Context Rule |
|---|---|---|
| global | worldwide, international | International often involves relations between nations. |
| modern | contemporary, present-day | Contemporary is more formal. |
| national | nationwide, domestic | Domestic often contrasts with foreign/international. |
| local | community-based, neighborhood | Neighborhood is small-area and less formal. |
| time | leisure hours, eras | Eras is for historical periods. |
| many | numerous, a large number of | Both are formal-safe alternatives. |
| more | greater, additional | Greater is often used for amount/degree. |
| popular (liked by many) | widespread, prevalent | Use for acceptance/adoption in groups. |
| common (occurs often) | frequent, widespread, ubiquitous | Common != popular. |

## Accuracy Filter for IELTS
### Direct safe substitutions
- global -> worldwide, international
- modern -> contemporary
- many -> numerous
- more -> additional
- common -> frequent

### Context-dependent substitutions
- national -> domestic, nationwide
- local -> community-based
- popular -> prevalent, widespread
- time -> leisure hours, eras

### Use carefully
- ubiquitous is strong and spelling-sensitive.
- popular means liked; common means frequent.

## Example Sentences
- Climate change is a worldwide concern requiring international cooperation.
- Contemporary workplaces demand both technical and interpersonal skills.
- Domestic production can reduce dependence on imports.
- A large number of families rely on community-based services.
- Traffic congestion is common in large cities, but it is not popular.
