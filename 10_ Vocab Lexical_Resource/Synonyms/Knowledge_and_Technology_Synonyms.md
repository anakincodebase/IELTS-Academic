# Knowledge and Technology Synonyms

## Core Mapping (IELTS-Safe)
| Core Word | Better Alternatives | Context Rule |
|---|---|---|
| technology | innovation, digital tools | Digital tools works for apps/AI/platforms. |
| education | schooling, learning | Schooling is school-focused; learning is broader. |
| experience (noun) | practical knowledge, expertise | Expertise signals high-level mastery. |
| skills | abilities, competencies, prowess | Competencies is job/business-oriented; prowess is very strong skill. |
| information | data, knowledge | Data means raw facts/figures; knowledge means understood information. |

## IELTS-Safe Usage Notes
- Use innovation for new methods/products, not for all technology.
- Use expertise when describing advanced professional competence.
- Use competencies in workplace or employability contexts.

## Accuracy Filter for IELTS
### Direct safe substitutions
- technology -> digital tools
- education -> learning
- skills -> abilities
- information -> data, knowledge

### Context-dependent substitutions
- experience -> practical knowledge (hands-on context)
- experience -> expertise (advanced level)
- skills -> competencies (job market/policy writing)
- skills -> prowess (high-performance contexts)

### Use carefully
- data may take plural verb in formal writing (data are), but singular usage is common in modern style; be consistent.
- prowess is strong and stylistically marked.

## Example Sentences
- Digital tools can improve access to remote learning.
- Schooling quality often depends on teacher preparation and class size.
- Practical knowledge is essential in technical professions.
- Employers increasingly value communication competencies.
- Reliable data helps governments design evidence-based policy.
