## Academic/Professional Words (80 words)

These dominate **Sections 3-4** (university lectures and seminars).

### A-B
- **accommodation** /ə-kɒm-ə-ˈdeɪ-ʃən/ → where you stay. "Accommodation for students includes dormitories."
- **achievement** /ə-ˈtʃiːv-mənt/ → accomplishment. "Her achievement in mathematics was exceptional."
- **analysis** /ə-ˈnæl-ə-sɪs/ → detailed examination. "Statistical analysis shows..."
- **application** /ˌæp-lɪ-ˈkeɪ-ʃən/ → formal request. "Submit your application by Friday."
- **assessment** /ə-ˈses-mənt/ → evaluation. "Your assessment will be based on..."
- **assignment** /ə-ˈsaɪn-mənt/ → task given to complete. "The assignment is due next week."
- **atmosphere** /ˈæt-məs-fɪər/ → the air; mood. "The atmosphere in the lab was intense."
- **attendance** /ə-ˈten-dəns/ → showing up. "Attendance is mandatory."
- **attitude** /ˈæt-ɪ-tuːd/ → way of thinking. "A positive attitude helps."
- **authority** /ɔː-ˈθɒr-ə-ti/ → official power. "The authority approved the project."
- **availability** /ə-ˌveɪ-lə-ˈbɪl-ə-ti/ → being available. "Check the availability of rooms."

## Common Errors in IELTS Listening

### Spelling Traps (Words Often Misspelled)

| Correct | Common Mistake | Context |
|---|---|---|
| accommodation | accomodation (one 'c') | "Accommodation provided" |
| commitment | committal | "Commitment is important" |
| occurred | occured (one 'c') | "An error occurred" |
| convenient | convient | "Convenient time?" |
| necessary | neccessary (extra 'c') | "Training necessary" |
| separate | seperate | "Separate groups" |
| maintenance | maintainance | "Maintenance required" |
| occurrence | occurence | "Regular occurrence" |
| liaison | liason | "Liaison officer" |
| questionnaire | questionair | "Complete the questionnaire" |
| medieval | mediaeval | "Medieval history" |
| privilege | priviledge | "Staff privilege" |
| threshold | threshhold | "Cross the threshold" |
| rhythm | rythm | "Musical rhythm" |
| cemetery | cemetary | "Historic cemetery" |

### Homophone Traps (Sound the Same, Different Spelling)

| Words | Meaning | Context |
|---|---|---|
| there / their / they're | direction / possession / they are | "Put it there" vs "Their books" |
| to / too / two | direction / also / number | "Go to class" vs "Too much" vs "Two students" |
| right / write | correct / mark on paper | "Right answer" vs "Write your name" |
| break / brake | rest / stopping device | "Take a break" vs "Apply the brake" |
