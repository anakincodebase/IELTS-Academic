# Listening Spelling Predictor: 200+ IELTS Words with Variants

## How to Use This List

**Before Listening Practice:**
1. Scan this list for words relevant to the topic
2. Pre-memorize spelling (not pronunciation)
3. During listening: Focus on RECOGNIZING, not spelling from scratch

**Spelling Trap Examples:**
```
Hear: "The facilit..." → Is it facility or facility? (Common Spelling confusion)
Pre-knowing: "facility" (not "facilility") → Write confidently
```
