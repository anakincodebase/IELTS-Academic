# Adverbs Quick Revision

## 60-Second Placement Rules
- Frequency: before main verb (often go), after be (is usually).
- Manner: usually after object (solve problems efficiently).
- Sentence adverbs: usually clause openers (Consequently, ...).

## Order Rule
When multiple adverbials appear:
- Manner -> Place -> Time

Example:
- They solved the issue quickly in the office yesterday.

## High-Value IELTS Adverbs
- stance: arguably, evidently, apparently
- logic: consequently, therefore, nevertheless
- degree: significantly, substantially, marginally

## Common Traps
- adjective/adverb confusion: drive careful -> drive carefully
- only placement ambiguity: only students passed vs students only passed
- overstacking adverbs in one sentence

## Task 1 Fast Bank
- rose significantly
- fell slightly
- remained relatively stable
- fluctuated considerably

## 20-Second Final Scan
- Right adverb form?
- Right position?
- Clear sentence scope?
- Any unnecessary adverb clutter?
