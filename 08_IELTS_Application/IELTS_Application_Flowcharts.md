# IELTS Application Flowcharts

## Why This Note
Use these flowcharts for fast revision before practice tests and exam day.

## 1) Full IELTS Prep Pipeline
```mermaid
flowchart LR
    A[Set Target Band] --> B[Diagnostic Test]
    B --> C{Weakest Module?}
    C -->|Writing| D[Task 1 + Task 2 System]
    C -->|Speaking| E[Part 1-2-3 Framework]
    C -->|Reading| F[Skim-Scan-Locate Drill]
    C -->|Listening| G[Prediction + Trap Control]
    D --> H[Timed Practice]
    E --> H
    F --> H
    G --> H
    H --> I[Error Log + Fix]
    I --> J[Mock Test]
    J --> K{Target Stable?}
    K -->|No| C
    K -->|Yes| L[Exam Ready]
```

## 2) Writing Task 1 Report Builder
```mermaid
flowchart TD
    A[Read Prompt + Units + Time] --> B[Identify Chart Type]
    B --> C[Find 2-4 Main Features]
    C --> D[Opening: One Sentence Paraphrase]
    D --> E[Overview: Biggest Trends]
    E --> F[Body 1: Highest-Lowest-Middle Group]
    F --> G[Body 2: Remaining Key Comparisons]
    G --> H[Final Check: Tense, Numbers, Clarity]
```

## 3) Writing Task 2 Essay Builder (POER)
```mermaid
flowchart TD
    A[P: Planning 2-3 min] --> B[O: Organize 2 min]
    B --> C[E: Evaluate Logic 1 min]
    C --> D[R: Revision Plan 1 min]
    D --> E[Intro: 3 Sentences]
    E --> F[Body 1: Topic-Explain-Example]
    F --> G[Body 2: Topic-Explain-Example]
    G --> H[Conclusion: 2 Sentences]
    H --> I[Backward Proofread]
```

## 4) Speaking Flow (Part 1, 2, 3)
```mermaid
flowchart LR
    A[Hear Question] --> B{Which Part?}
    B -->|Part 1| C[Direct Answer + Reason + Mini Example]
    B -->|Part 2| D[1-min Plan -> Story Flow -> 1-2 min Talk]
    B -->|Part 3| E[Opinion -> Reason -> Example -> Balance]
    C --> F[Grammar Accuracy]
    D --> F
    E --> F
    F --> G[Natural Pace + Clear Pronunciation]
```

## 5) Reading Answer Flow
```mermaid
flowchart TD
    A[Read Question First] --> B[Identify Keyword + Paraphrase]
    B --> C[Skim Passage Structure]
    C --> D[Scan Target Area]
    D --> E[Match Meaning, Not Just Words]
    E --> F[Check Word Limit/Grammar]
    F --> G[Final Transfer Check]
```

## 6) Listening Safety Flow
```mermaid
flowchart TD
    A[Before Audio: Read Questions] --> B[Predict Word Type]
    B --> C[Listen for Synonyms/Paraphrase]
    C --> D[Write Fast Placeholder]
    D --> E[Catch Traps: But/However/Actually]
    E --> F[Spelling + Plural Check]
    F --> G[Answer Transfer Accuracy]
```

## 7) Exam-Day Time Map
```mermaid
flowchart LR
    A[Task 2: 40 min] --> B[Task 1: 20 min]
    B --> C[Reading: 60 min]
    C --> D[Listening: 30 min + transfer time]
    D --> E[Speaking Slot: Calm Delivery]
```

## How to Use
1. Read one flowchart before each practice session.
2. Follow it while writing or speaking.
3. Mark the step where you failed.
4. Drill only that step for 2-3 days.
