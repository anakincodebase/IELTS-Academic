# Personal Error Tracking Template

## How to Use This Template

**1. After Each Writing Practice Session:**
- Copy ONE completed row (below)
- Fill in every column immediately while error is fresh
- Identify pattern (don't just record error)

**2. After Each Speaking Practice Session:**
- Record errors you noticed or corrected
- Include what you did to recover (if applicable)

**3. Weekly Review (Every Sunday):**
- Scan your error log for repeated patterns
- Count: Which 3 error types appear most?
- Those 3 become next week's focus (see drills below)

**4. Monthly Progress Check:**
- Count total errors: Did the number decrease?
- Pattern variety: Are you making DIFFERENT errors now (signs of progress)?
- Severity: Are Band-6-level errors decreasing, replaced by Band-7-level subtleties?

## Error Categories (For Column 5)

Use these short codes:

1. **SVA** = Subject-Verb Agreement
2. **PronAg** = Pronoun Agreement
3. **Tense** = Tense Consistency
4. **Article** = A/An/The usage
5. **Prep** = Preposition usage
6. **WordForm** = Noun/Verb/Adj/Adv precision
7. **Homophone** = Their/There/They're, etc.
8. **Fragment** = Incomplete sentence
9. **RO** = Run-On sentence
10. **Comma** = Comma splice
11. **Collocation** = Wrong word choice/combination
12. **Spelling** = Misspelled word

## Drill Templates: Use These to Fix Your Top 3 Errors

### Drill Format (for any error type)

**Step 1: Error Recognition (5 min)**
Read 10 sentences; identify IF/WHEN the error appears:
```
1. "The data shows a pattern."
   ✓ Correct / ✗ Error: [category: _____]

2. "The research team are working."
   ✓ Correct / ✗ Error: [category: _____]
```

**Step 2: Error Correction (5 min)**
Rewrite each error correctly; explain why:
```
1. "The data shows a pattern."
   → "The data demonstrate a pattern."
   Because: Data is plural; "demonstrate" matches.
```

**Step 3: Production (10 min)**
Write 5 sentences using this grammar rule correctly:
```
1. [Your own sentence using the rule correctly]
2. [Your own sentence using the rule correctly]
...
```
