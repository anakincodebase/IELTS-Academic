# Sentence Structure Errors: Fragments, Run-Ons & Comma Splices

## Why Sentence Structure Blocks Band 6→7

**IELTS Examiner Perspective:**
- Sentence boundaries (where sentences begin/end) reveal writer's grammatical control
- Band 6 often has fragments (incomplete sentences) OR run-ons (sentences merged improperly)
- Band 8 maintains consistent sentence structure with clear boundaries
- These errors create confusion and seem unprofessional

**Example Band 6→8:**
```
Band 6: "The research showed results. Because the methodology was rigorous." ✗
        (Fragment: incomplete dependent clause)

Band 8: "The findings proved significant because the methodology incorporated rigorous control mechanisms." ✓
        (Complete sentence: independent clause + dependent clause properly joined)
```

---

## THE 3 MAJOR SENTENCE ERRORS

### ERROR 1: SENTENCE FRAGMENTS (Incomplete Sentences)

**Definition**: A sentence that lacks either:
1. An independent clause (subject + verb), OR
2. A complete thought

**Fragment Type A: Missing Main Verb**

```
✗ Band 6: "The methodology rigorous and systematic."
         (Missing verb "is"; has only descriptor)

✓ Band 8: "The methodology was rigorous and systematic."
         (Complete: subject + verb + descriptors)
```

**Fragment Type B: Dependent Clause Only**

```
✗ Band 6: "Although the sample was limited. The conclusions remained valid."
         (Dependent clause (although...) presented as sentence)

✓ Band 8: "Although the sample was limited, the conclusions remained valid."
         (Dependent clause joined to independent clause with comma)
```

**Fragment Type C: Verbal Phrase Only (Gerund/Participle)**

```
✗ Band 6: "Analyzing the data. Researchers discovered patterns."
         (Gerund phrase "analyzing..." without main clause)

✓ Band 8: "Upon analyzing the data, researchers discovered patterns."
         (Prepositional phrase + main clause)
```

**Fragment Type D: Prepositional Phrase Only**

```
✗ Band 6: "In the research facility. Where experiments occurred."
         (Prepositional phrase only; no main clause)

✓ Band 8: "In the research facility, where experiments occurred, rigorous protocols governed all procedures."
         (Prepositional phrase + main clause with embedded clause)
```

### How to Fix Fragments:

1. **Attach to Previous Sentence** (if related):
   ```
   Before: "The study was comprehensive. Including 500 participants."
   After: "The study was comprehensive, including 500 participants."
   ```

2. **Add Missing Verb**:
   ```
   Before: "The results significant and measurable."
   After: "The results were significant and measurable."
   ```

3. **Combine with Independent Clause**:
   ```
   Before: "Although time was limited. The analysis continued."
   After: "Although time was limited, the analysis continued."
   ```

---

### ERROR 2: RUN-ON SENTENCES (Two Independent Clauses Merged)

**Definition**: Two complete sentences joined with no punctuation or inappropriate connecting word

**Type A: Fused Sentence (No Connector)**

```
✗ Band 6: "The research showed significant results researchers were satisfied."
         (Two independent clauses with no connector; called "comma fault")

✓ Band 8: "The research demonstrated significant results, and researchers validated all findings."
         (Proper conjunction: "and")
```

**Type B: Comma Splice (Only Comma, No Conjunction)**

```
✗ Band 6: "The methodology was rigorous, it contained clear procedures."
         (Two independent clauses joined only by comma; needs conjunction or period/semicolon)

✓ Band 8: "The methodology was rigorous; it contained clear procedural frameworks."
         (Semicolon joins related independent clauses)
```

**Type C: Dependent Clause Marker Missing**

```
✗ Band 6: "The results were positive the analysis was thorough."
         (Appears to merge two sentences; needs "because")

✓ Band 8: "The results were positive because the analysis was thorough."
         (Dependent clause marker "because" shows relationship)
```

### How to Fix Run-Ons:

**Method 1: Add Conjunction** (FANBOYS: For, And, Nor, But, Or, Yet, So)
```
Before: "The study proved validity the team moved forward."
After: "The study proved validity, so the team moved forward."
```

**Method 2: Use Semicolon** (for closely related independent clauses)
```
Before: "The methodology was sound it ensured reliability."
After: "The methodology was sound; it ensured reliability."
```

**Method 3: Split into Two Sentences**
```
Before: "The research took three years participants completed all phases."
After: "The research took three years. Participants completed all phases."
```

**Method 4: Create Dependent Clause**
```
Before: "The sample was large the results were generalizable."
After: "Because the sample was large, the results were generalizable."
```

---

### ERROR 3: COMMA SPLICE (Specific Type of Run-On)

**Definition**: Two independent clauses joined by only a comma (without conjunction)

**Specific Pattern:**

```
✗ Pattern: "Independent Clause 1 , Independent Clause 2"

Examples:
✗ "The findings were significant, they published immediately."
✗ "The protocol was established, compliance improved."
✗ "Data collection ended, analysis began."
```

### Comma Splice Fixes (4 Options):

**Option 1: Add Conjunction (becomes compound sentence)**
```
Before: "The study ended, researchers analyzed results."
After: "The study ended, and researchers analyzed results."
```

**Option 2: Use Semicolon (maintains relationship)**
```
Before: "The study ended, researchers analyzed results."
After: "The study ended; researchers analyzed results."
```

**Option 3: Use Colon (shows cause-effect)**
```
Before: "The methodology proved effective, compliance rates improved."
After: "The methodology proved effective: compliance rates improved significantly."
```

**Option 4: Create Dependent Clause**
```
Before: "The protocol was clear, implementation succeeded."
After: "Because the protocol was clear, implementation succeeded."
```

## IDENTIFYING SENTENCE BOUNDARIES

### Key Question: Is This a Complete Sentence?

**Test 1: Independent Clause Check**
```
Question: "Does this clause have BOTH subject + verb AND express complete thought?"

✓ "The research demonstrated results." 
  Subject: research | Verb: demonstrated | Complete? YES

✗ "Because the methodology was rigorous."
  Subject: methodology | Verb: was | Complete? NO (dependent clause—"because" makes it incomplete)
```

**Test 2: Dependent Marker Check**
```
Dependent markers (subordinating conjunctions):
because, although, when, while, since, if, unless, whenever, after, before, until

Sentence with dependent marker alone = NOT COMPLETE = FRAGMENT

✗ "Although the research was comprehensive."
   (Has dependent marker "although" + subject + verb, but NOT complete)

✓ "Although the research was comprehensive, the sample remained small."
   (Dependent clause + independent clause = complete)
```

## 20 COMMON SENTENCE STRUCTURE ERRORS

### Rank 1-5: Fragments (Most Common)

1. **Dependent Clause Only**
   ```
   ✗ "If the data were analyzed properly."
   ✓ "If the data were analyzed properly, insights would emerge."
   ```

2. **Gerund Phrase Only**
   ```
   ✗ "Analyzing the results. They discovered patterns."
   ✓ "Upon analyzing the results, they discovered patterns."
   ```

3. **Missing Main Verb**
   ```
   ✗ "The study rigorous and systematic."
   ✓ "The study was rigorous and systematic."
   ```

4. **Prepositional Phrase Only**
   ```
   ✗ "In the laboratory. The experiments commenced."
   ✓ "In the laboratory, the experiments commenced."
   ```

5. **Because/Since Fragment**
   ```
   ✗ "The results were significant. Because the methodology was sound."
   ✓ "The results were significant because the methodology was sound."
   ```

### Rank 6-10: Comma Splices (Most Common Run-On)

6. **Basic Comma Splice**
   ```
   ✗ "The study ended, researchers analyzed findings."
   ✓ "The study ended; researchers analyzed findings." OR
   ✓ "The study ended, and researchers analyzed findings."
   ```

7. **Comma Splice with Pronoun**
   ```
   ✗ "The protocol was established, it ensured consistency."
   ✓ "The protocol was established; it ensured consistency."
   ```

8. **Comma Splice with Multiple Subjects**
   ```
   ✗ "The team conducted research, they published results."
   ✓ "The team conducted research and published results."
   ```

9. **Comma Splice Before "However"**
   ```
   ✗ "The sample was large, however, limitations existed."
   ✓ "The sample was large; however, limitations existed."
   ✓ "The sample was large, but limitations existed."
   ```

10. **Comma Splice Before "Therefore"**
    ```
    ✗ "The methodology was sound, therefore, conclusions were justified."
    ✓ "The methodology was sound; therefore, conclusions were justified."
    ```

### Rank 11-20: Complex Errors (Band 6→7)

| Error | Correction | Category |
|---|---|---|
| "The study showed results. Which were significant." | "The study showed results, which were significant." | Fragment using relative clause |
| "Factors included timing, resources, and expertise. Making success possible." | "Factors including timing, resources, and expertise made success possible." | Dangling modifier |
| "The experiment ran for three months, data collection occurred throughout." | "The experiment ran for three months; data collection occurred throughout." | Comma splice |
| "Data demonstrated patterns, furthermore, correlations emerged." | "Data demonstrated patterns; furthermore, correlations emerged." | Semicolon before conjunctive adverb |
| "The policy was enacted. In 2015 when support existed." | "The policy was enacted in 2015, when support was abundant." | Dangling time phrase |
| "Although the sample was limited the conclusions were valid." | "Although the sample was limited, the conclusions were valid." | Missing comma after dependent clause |
| "The findings suggested recommendations were needed, although limitations remained." | "The findings suggested recommendations were needed, although limitations remained." | Correct, but make "suggestions" more concrete |
| "The analysis completed, results were interpreted quickly." | "Upon analysis completion, results were interpreted quickly." OR "When analysis completed, results were interpreted quickly." | Dangling participle phrase |
| "Meeting the objectives the team continued forward." | "Meeting the objectives, the team continued forward." (add comma) | Dangling participle |
| "The protocol clear and systematic implementation proceeded." | "With the protocol clear and systematic, implementation proceeded." | Missing article + misfiring modifier |

## DIAGNOSTIC FLOWCHART

```
START: Examine sentence

Q1: Does the sentence have an independent clause (subject + verb)?
├─ NO → FRAGMENT (add main clause)
│
└─ YES → Continue

Q2: Is there a dependent marker (because, although, when, etc.)?
├─ YES → Check: Is there ALSO an independent clause after the marker?
│   ├─ NO → FRAGMENT (add independent clause after the dependent clause)
│   └─ YES → Continue
│
└─ NO → Continue

Q3: How many independent clauses are in the sentence?
├─ ONE → Sentence is complete ✓
│
└─ TWO OR MORE → Check junction
    ├─ Joined by: period (.) → Two sentences ✓
    ├─ Joined by: conjunction (and, but, or) with comma → Compound sentence ✓
    ├─ Joined by: semicolon (;) → Complex sentence ✓
    ├─ Joined by: comma only → COMMA SPLICE ✗ (fix: add conjunction or semicolon)
    └─ Joined by: nothing → FUSED SENTENCE ✗ (fix: add conjunction, semicolon, or period)
```

## SELF-DIAGNOSTIC TEST

**Identify the error (F = Fragment, CS = Comma Splice, RO = Run-On, ✓ = Correct):**

1. "The study demonstrated results. Because the methodology was rigorous."
   Answer: **FRAGMENT** ("Because..." is dependent clause only)

2. "The team analyzed data, they published findings."
   Answer: **COMMA SPLICE** (Two independent clauses joined by only comma)

3. "Although the sample was limited, the conclusions remained valid."
   Answer: **✓ CORRECT** (Dependent clause + independent clause)

4. "The protocol established clear procedures implementation succeeded."
   Answer: **FUSED SENTENCE OR RUN-ON** (Two independent clauses, no connector)

5. "Data collection lasted six months; analysis took three."
   Answer: **✓ CORRECT** (Two independent clauses joined by semicolon)

## 7-DAY SENTENCE STRUCTURE MASTERY

### Day 1: Fragment Recognition
Read 20 test sentences; identify which are fragments (F) vs complete (C):
```
1. "Although the research ended." (F)
2. "The research ended." (C)
3. "Analyzing the data carefully." (F)
4. "After the analysis concluded, results emerged." (C)
```

### Day 2: Fragment Fixes
Convert 15 fragments to complete sentences:
```
Before: "Because the methodology was sound."
After: "The results were significant because the methodology was sound."
```

### Day 3: Comma Splice Detection
Read 20 sentences; identify comma splices (CS) vs correct (✓):
```
1. "The study ended, researchers began analysis." (CS)
2. "The study ended; researchers began analysis." (✓)
```

### Day 4: Comma Splice Fixes (3 Methods)
Fix 15 comma splices using:
- Method 1 (add conjunction)
- Method 2 (semicolon)
- Method 3 (dependent clause)

### Day 5: Run-On & Fusion Detection
Read 25 sentences; identify run-ons (RO) vs comma splices (CS) vs correct (✓)

### Day 6: Comprehensive Sentence Correction
Take 30 sentences with mixed errors (fragments, comma splices, run-ons); correct all

### Day 7: Production Writing
Write 5-paragraph essay with 100% correct sentence boundaries (no fragments, comma splices, or run-ons)

## QUICK REFERENCE: HOW TO FIX EACH ERROR

| Error Type | Signs | Quick Fix |
|---|---|---|
| Fragment | Dependent marker (because, although, when) without independent clause | Add independent clause after |
| Fragment | No main verb | Add verb |
| Fragment | Gerund/participle phrase only | Add "Upon [phrase], [main clause]" |
| Comma Splice | Two independent clauses + only comma | Add conjunction (and, but, so) OR use semicolon |
| Run-On (Fused) | Two independent clauses + no connector | Add conjunction, semicolon, period, or create dependent clause |
| Run-On (Comma Splice variant) | Comma before "however, therefore, furthermore" | Use semicolon before these words, not comma |
